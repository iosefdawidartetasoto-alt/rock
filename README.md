# 🎸 Shattered Riffs – Landing Page (HTML & CSS)

## Overview

A responsive landing page for a fictional rock band built with semantic HTML5 and CSS3. The goal is to create a visually engaging site that reflects the band's identity and promotes their upcoming tour.

---

## Project Structure

```
├── assets/
│   ├── css/style.css
│   ├── icons/logo.ico
│   ├── img/img-1.jpg … img-10.jpg
│   └── js/main.js
├── index.html
└── README.md
```

---

## Sections

| Section | Description |
|---|---|
| **Hero** | Main visual with band name and CTA |
| **Events** | Tour dates in a table |
| **About** | Band bio and influences |
| **Media** | Photo gallery grid |
| **Footer** | Copyright and social links |

---

## Requirements

**HTML**
- Semantic tags: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- At least one unordered list (influences, social links, etc.)
- Table for events (extra points)

**CSS**
- Custom colors, typography and spacing
- Flexbox and/or Grid for layout
- Media queries for desktop (>1024px) and mobile

**JavaScript** *(optional)*
- Mobile hamburger menu toggle
- Dynamic year in footer

---

## How to Run

1. Clone or download the project
2. Open `index.html` in your browser — no installation needed
